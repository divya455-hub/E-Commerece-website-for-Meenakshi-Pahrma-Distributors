# E-Commerce-website-for-Meenakshi-Pharma-Distributors
## 📌 Project Overview
This is a full-stack e-commerce website developed for managing pharmaceutical products.  
It includes both customer and admin functionalities.

## 🚀 Features
- User Registration & Login
- Product Browsing
- Add to Cart & Checkout
- Admin Dashboard
- Product Management (Add/Edit/Delete)
- Image Upload via Admin Panel

## 🛠️ Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB

## 📸 Note
Product images are dynamically uploaded through the admin panel.  
Images in the repository are only placeholder/demo images.
